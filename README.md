# Переводчик защищенных видео
1. Открыть код файла final.py
2. Вписать свой яндекс апи токен (доступ к яндекс диску)
3. Запустить файл
4. В открывшемся браузере использовать расширение впн, если нужно
5. Авторизоваться, перейти на нужное видео.
6. Перед запуском выставить нужное качество
7. Запустить видео, нажать в консоли энтер
8. Прогрузить видео целиком
9. Ожидать перевода
10. В конце получится mp4 файл с переводом. Время полного цикла программы около 5-7 минут
